# Java Class - Training Course

## Spring Security Demo

#### Author: Herman Rabinkin

This repository contains examples which can help to explain and understand basic Spring Security configuration
Project is focused only on providing the most basic and common configuration options related to servlet based web
applications.

It contains username/password based authentication using JDBC user service.
CSRF configured and token transmission implemented in different ways - JS, Thymeleaf, request params as well as
headers

Some code in configuration class is commented out and committed in order to provide examples of different ways of
configuration
